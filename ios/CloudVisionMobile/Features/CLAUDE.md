# Features/

Feature areas, one subdirectory per major feature. Each holds its own Codable models, service wrappers, and views.

## Subdirectories

| Directory | What | When to read |
|-----------|------|--------------|
| `Locate/` | F1 Endpoint Locator — find which switch/port a MAC, IP, or hostname is connected to | Modifying endpoint search, result rendering, recent searches |
| `Switches/` | F2 Switch & Port Inspector (first-light) + F3 Device Health Summary | Modifying device list, device health, port inspector, alerts/events |
